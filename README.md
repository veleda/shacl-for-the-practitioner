# shacl-for-the-practitioner
Contains the shapes as described in Ch. 6 Core Constraint Components from [_SHACL for the Practitioner_](https://veronahe.wordpress.com/shacl-for-the-practitioner/).

SHACL for the Practitioner is a comprehensive guide to mastring SHACL, offering readers the knowledge and tools needed to navigate this fascinating domain. It is a valuable resource for anyone looking to harness the power of RDF and SHACL to build the next generation of data systems.
